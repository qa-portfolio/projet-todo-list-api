# Projet: application de prises de notes avec Express et une API Rest

## Fonctionnalités attendues:

- Ajouter de nouvelles tâches
- Modifier les tâches existantes
- Supprimer des tâches
- Lister toutes les tâches
- Marquer les tâches comme terminées

## Stack front-end

- HTML
- CSS
- JavaScript

## Stack Back-end

- Node.js
- Express

## Méthodologies de Test

### Postman

- Validation complète de l'API avec Postman
- Tests automatisés avec Newman
- Génération de rapports avec Newman

### CI/CD

- Intégration continue sur Github avec Github actions
- Exécution de la pipeline à chaque ajout de code

## Installation

- Télécharger / Cloner le répertoire
- Naviguer dans le projet
- Installation des dépendences avec la commande: npm install
- Lancer le projet avec la commande: npm start
- Application disponible sur http://localhost:3000
